## 개요

---

### 개발 동기

> -   e스포츠 시장이 활성화 되면서 다양한 e스포츠 프로 경기들이 있지만, 어떤 대회들을 잘 알려져 있지 않거나 시간대가 애매해서 잘 모르는 경우가 있다. 따라서, 해당 e스포츠 대회들에 대한 정보를 한 데 모으고, 사용자의 선호도에 맞춘 달력을 보여줌으로써 e스포츠 대회에 대한 전반적인 관심과 흥미를 유발하고자 정하였다.

<br>
<br>

## 설치 및 실행 방법

---

### 설치

1. Releases에 있는 배포 버전 중 OS에 맞는 배포 버전 zip 파일 다운로드
2. zip파일 압축 해제 후 어플리케이션 실행  
### Windows
zip 압축 풀기 확장자 실행 후, `e-sports-scheduler.exe` 실행  
### Linux
```
 $ unzip e-sports-scheduler-linux-x64.zip
 $ cd e-sports-scheduler-linux-x64
 $ ./e-sports-scheduler
```
